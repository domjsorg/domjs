# DomJS
domjs repository

## Project setup
```
npm install
```

### Compiles and browser-reloads for development (All files DomJS Library + Mambo Tools & Mambo UI + Demo)
```
npm run dev
```

### Compiles and minifies for production (Only DomJS Library)
```
npm run build
```

### Show Demo web
```
npm run start
```
